📖 Endpoints Disponibles

POST /api/track

Envía un punto GPS (requiere JWT)

{ "latitude": 40.4168, "longitude": -3.7038, "altitude": 667.5, "timestamp_utc": "2024-01-15T14:30:00Z", "speed": 15.3 }

GET /api/tracks/:route_name

Obtiene todos los puntos de una ruta

  • Parámetros: ?format=geojson (opcional)
  • Ejemplo: /api/tracks/ruta-madrid

GET /api/tracks/:route_name/latest

Obtiene el último punto de una ruta (posición actual)

  • Parámetros: ?format=geojson (opcional)
  • Ejemplo: /api/tracks/ruta-madrid/latest

🗺️ Ejemplo con Leaflet

// Mostrar todos los puntos de una ruta
fetch('/api/tracks/ruta-madrid?format=geojson')
  .then(r => r.json())
  .then(geojson => {
    L.geoJSON(geojson).addTo(map);
  });

// Actualizar posición en tiempo real
function updatePosition() {
  fetch('/api/tracks/ruta-madrid/latest?format=geojson')
    .then(r => r.json())
    .then(point => {
      marker.setLatLng([
        point.geometry.coordinates[1],
        point.geometry.coordinates[0]
      ]);
    });
}

setInterval(updatePosition, 5000); // Actualizar cada 5 segundos
